## Definción 💡 ¿Qué es Javascript?: 
> Es un lenguaje de programación desarrollado inicialmente para la web
> Es un lenguaje para FRONTEND (lo que vemos en pantalla)
> Es un lenguaje para la interactividad (que el usuario pueda de algún modo disparar alguna acción en su pantalla)

## 💡 ¿Qué podemos hacer con Javascript?: 
> Podemos re escribir DOM
> Podemos modificar características o atributos de un elemento
> Podemos modificar el contenido de un elemento
> Podemos crear elementos a pedido
