# Funciones en javascript

## 💡 ¿qué es una función?
> Una función es un bloque de código para realizar una tarea específica
> Este bloque de código que puede tener varias acciones se va a almacenar en memoria
> Es como un "mecanismo" reutilizable que guarda instrucciones

## 💡 ¿para qué sirve una función?
[] Organizar el código en partes pequeñas y claras
[] Reutilizar código sin escribirlo nuevamente
[] Evitar errores y facilitar cambios
[] Lograr que sea más fácil de leer y de mantener

## 💡 partes de una función 
1. Palabra clave para declararla: function
2. Nombre de una función *
3. Cuerpo de la función (entre llaves) 

    function nombreFuncion()  
    {  
        // bloque de código
    }  

> En la declaración de una función hacemos que exista, que se guarde en memoria. 
> Pero no se ejecuta   

## 💡 Ejecución de una función 

> Para ejecutar una función debemos invocarla, Debemos llamarla
> Una función se invoca por su nombre

    nombreFuncion()  

## Ejemplo

    //declaración
    function saludar()  
    {  
        console.log('hola')
    }  

    // invocación 
    saludar()
