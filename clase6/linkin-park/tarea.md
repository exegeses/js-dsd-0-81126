# Tarea:

> La idea es usar el Mouse sobre las imágenes miniatura cambien la imagen central (ampliada)

> Primero vamos a crear el archivo JavaScript
> Vamos a tener que ubicar 7 elementos dentro del documento (la imagen ampliada de las seis miniaturas)

## Desafío 
> crear una función llamada cambiarImagen()
> que reciba el nombre de la imagen que queremos reemplazar
> con cada una de las imágenes miniatura invocan a esta función pasándole el con el nombre de la imagen que queremos reemplazar
